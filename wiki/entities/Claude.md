## 概述

Claude 是由 [[Anthropic.md]] 開發的大型語言模型系列，提供 Claude.ai（網頁）、Claude Code（CLI）、Claude API 三種主要存取介面。

---

## 關鍵事實
- 支援 [[Claude-Skills.md]]：透過 Skills 系統，使用者可以固化工作流程與領域知識
- 支援 [[MCP-Model-Context-Protocol.md]]：透過 MCP 連接外部工具與服務
- Skills 在 Claude.ai、Claude Code、API 三個介面行為一致（可攜性設計）
- 組織層級：Admin 可將 skills 部署至整個 workspace（2025 年 12 月起支援）

---

## 在本知識庫的角色

Claude（Claude Code 介面）是本知識庫的 LLM Wiki Agent，負責撰寫、維護所有 wiki 頁面。
